## 🔧 Для заказчика - ЧТО ДЕЛАТЬ СЕЙЧАС:

### 1. ОБЯЗАТЕЛЬНО: Обновите приложение до версии 17

**В TestFlight:**
1. Откройте TestFlight
2. Найдите ваше приложение
3. Если есть обновление - нажмите **"Обновить"**
4. Убедитесь, что версия **1.0.0+17** или выше

**Если обновления нет:**
1. **Удалите приложение** с iPhone полностью
2. **Перезагрузите iPhone** (зажмите кнопку питания + кнопку уменьшения громкости)
3. **Установите приложение заново** из TestFlight
4. Попробуйте открыть

### 2. Проверьте интернет-соединение

**На iPhone:**
1. Откройте **Safari**
2. Введите в адресной строке: `https://milviar.ru`
3. Нажмите Enter

**Ожидаемый результат:**
- ✅ Должно открыться БЕЗ предупреждений
- ✅ Должно показать: `{"status":"ok","message":"API Server is running"}`

**Если НЕ открывается:**
- ❌ Проблема с DNS или оператором
- Попробуйте через Wi-Fi (должно работать)
- Попробуйте через VPN

### 3. Перезагрузите iPhone

**Простое решение, которое часто помогает:**
1. Зажмите кнопку питания + кнопку уменьшения громкости
2. Выберите **"Перезагрузить"**
3. Подождите полной перезагрузки
4. Попробуйте открыть приложение

### 4. Проверьте VPN

**На iPhone:**
- Настройки → VPN
- Если VPN включен - **отключите**
- Попробуйте снова

### 5. Проверьте ограничения

**На iPhone:**
- Настройки → Экранное время → Ограничения контента
- Убедитесь, что установка приложений разрешена
- Убедитесь, что нет ограничений на использование интернета

## 🚨 Возможные причины проблемы

### 1. Оператор блокирует домен

**Симптомы:**
- Wi-Fi работает, мобильный интернет не работает
- Safari не открывает https://milviar.ru через мобильные данные

**Решение:**
- Используйте VPN (например, Cloudflare WARP - бесплатный)
- Или используйте Wi-Fi

### 2. DNS не распространился

**Симптомы:**
- Safari не может найти milviar.ru

**Решение:**
- Подождите 24 часа
- Или используйте другой DNS:
  - Настройки → Wi-Fi → (i) рядом с сетью → DNS → Настроить DNS → Вручную
  - Добавьте: `8.8.8.8` и `8.8.4.4` (Google DNS)

### 3. Старая версия приложения

**Симптомы:**
- Приложение использует старый URL или настройки

**Решение:**
- Удалите приложение
- Установите версию 17 из TestFlight

### 4. Проблемы с сертификатом на устройстве

**Симптомы:**
- Предупреждение о безопасности в Safari

**Решение:**
- Проверьте дату и время на устройстве (должны быть правильными)
- Попробуйте через Wi-Fi

## 📱 Альтернативные решения

### Вариант 1: Использовать VPN

Если оператор блокирует домен:
1. Установите VPN приложение (например, Cloudflare WARP - бесплатный)
2. Включите VPN
3. Попробуйте открыть приложение

### Вариант 2: Использовать Wi-Fi

Если мобильный интернет не работает:
- Используйте Wi-Fi для работы приложения
- Это временное решение, пока не решим проблему с мобильными данными

### Вариант 3: Сменить DNS

На iPhone:
1. Настройки → Wi-Fi
2. Нажмите (i) рядом с сетью Wi-Fi
3. Прокрутите вниз до "DNS"
4. Нажмите "Настроить DNS"
5. Выберите "Вручную"
6. Добавьте: `8.8.8.8` и `8.8.4.4`
7. Сохраните
8. Попробуйте снова
