# 🔧 Правильная последовательность команд

## ⚠️ Важно: Выполняйте команды БЕЗ комментариев (#)

---

## Шаг 1: Перезапустите эмулятор

```bash
killall Simulator
sleep 3
open -a Simulator
```

Подождите, пока эмулятор полностью загрузится (появится главный экран).

---

## Шаг 2: Очистите и пересоберите проект

```bash
cd msngIosAndroidv2

# Очистите Flutter
flutter clean

# Получите зависимости (ВАЖНО: это создаст Generated.xcconfig)
flutter pub get

# Теперь можно установить Pods
cd ios
rm -rf Pods Podfile.lock
pod install
cd ..
```

---

## Шаг 3: Запустите приложение

```bash
flutter run
```

---

## 🔍 Если эмулятор все еще в состоянии Shutdown:

### Вариант 1: Перезагрузите эмулятор через команду

```bash
# Найдите ID эмулятора
xcrun simctl list devices | grep "iPhone 17 Pro"

# Перезагрузите его (замените DEVICE_ID на реальный ID)
xcrun simctl shutdown DEVICE_ID
xcrun simctl boot DEVICE_ID
```

### Вариант 2: Используйте другой эмулятор

```bash
# Посмотрите список доступных эмуляторов
xcrun simctl list devices available

# Запустите другой эмулятор (например, iPhone 15)
flutter run -d "iPhone 15"
```

### Вариант 3: Пересоздайте эмулятор через Xcode

1. Откройте Xcode
2. Window → Devices and Simulators
3. Выберите "iPhone 17 Pro"
4. Нажмите "Delete"
5. Создайте новый эмулятор:
   - Нажмите "+" внизу слева
   - Выберите iPhone 17 Pro
   - Нажмите "Create"
6. Запустите приложение

---

## ✅ После успешного запуска:

Приложение должно запуститься без ошибок. Разрешения будут запрашиваться при первом использовании функций:
- Звонок → запросит микрофон (и камеру для видеозвонка)
- Запись голосового сообщения → запросит микрофон
- Выбор изображения → запросит доступ к галерее
