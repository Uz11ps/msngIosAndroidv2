# Быстрое решение проблемы с мобильным интернетом

## Для заказчика - что сделать СЕЙЧАС:

### 1. Проверьте в Safari

Откройте Safari на iPhone и введите: `https://milviar.ru`

**Если открывается БЕЗ предупреждений:**
- ✅ DNS работает
- ✅ HTTPS работает
- Проблема в приложении - переходите к шагу 2

**Если НЕ открывается или есть предупреждение:**
- ❌ Проблема с DNS или оператором
- Попробуйте через Wi-Fi (должно работать)
- Или используйте VPN

### 2. Обновите приложение

**В TestFlight:**
1. Откройте TestFlight
2. Найдите ваше приложение
3. Если есть обновление - нажмите "Обновить"
4. Убедитесь, что версия **1.0.0+16** или выше

**Если обновления нет:**
1. Удалите приложение с iPhone
2. Перезагрузите iPhone
3. Установите приложение заново из TestFlight
4. Попробуйте войти

### 3. Перезагрузите iPhone

Простое решение, которое часто помогает:
- Зажмите кнопку питания + кнопку уменьшения громкости
- Выберите "Перезагрузить"
- Попробуйте снова

### 4. Проверьте VPN

**На iPhone:**
- Настройки → VPN
- Если VPN включен - отключите
- Попробуйте снова

### 5. Попробуйте через Wi-Fi

Если через Wi-Fi работает, а через мобильный интернет нет:
- Используйте Wi-Fi для работы приложения
- Или используйте VPN для обхода блокировки оператора

---

## Если ничего не помогает

**Попросите заказчика:**
1. Сделать скриншот экрана приложения при попытке входа
2. Сделать скриншот ошибки (если есть)
3. Проверить в Safari: открывается ли https://milviar.ru
4. Отправить вам эту информацию

**Возможные причины:**
- Оператор блокирует домен milviar.ru
- DNS не распространился на устройстве заказчика
- Старая версия приложения
- Проблемы с сертификатом на устройстве

---

## Проверка для вас

**Убедитесь, что:**
1. ✅ HTTPS работает: `curl -I https://milviar.ru`
2. ✅ SSL сертификат валидный: `openssl s_client -connect milviar.ru:443`
3. ✅ Node.js приложение работает: `curl http://localhost:3000`
4. ✅ Nginx проксирует правильно: проверьте конфигурацию

**Если все работает на сервере:**
- Проблема на стороне заказчика (DNS, оператор, версия приложения)
