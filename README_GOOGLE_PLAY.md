# Публикация в Google Play - Краткое руководство

## 📋 Что нужно сделать

### 1. Подготовка (5-10 минут)
- [ ] Создать аккаунт разработчика на https://play.google.com/console ($25)
- [ ] Настроить подпись приложения (используйте `setup_keystore.bat` или `setup_keystore.sh`)
- [ ] Подготовить материалы (иконка, скриншоты, описание)

### 2. Сборка (5 минут)
```bash
flutter build appbundle --release
```

### 3. Загрузка в Google Play Console (30-60 минут)
- Создать приложение
- Загрузить материалы
- Загрузить App Bundle
- Заполнить информацию
- Отправить на проверку

### 4. Ожидание (1-3 дня)
- Google проверит приложение
- Вы получите email о результате

## 📁 Файлы инструкций

- **GOOGLE_PLAY_PUBLICATION_GUIDE.md** - Подробная пошаговая инструкция
- **QUICK_START_GOOGLE_PLAY.md** - Быстрый старт
- **setup_keystore.bat** / **setup_keystore.sh** - Скрипты для создания ключа подписи
- **privacy_policy_template.html** - Шаблон политики конфиденциальности

## ⚠️ Важно

1. **Сохраните ключ подписи!** Без него вы не сможете обновлять приложение
2. **Политика конфиденциальности обязательна** - разместите на сайте или GitHub Pages
3. **Минимум 2 скриншота** требуются для публикации
4. **Иконка 512x512** обязательна

## 🚀 Быстрый старт

1. Прочитайте `QUICK_START_GOOGLE_PLAY.md`
2. Следуйте инструкциям
3. При необходимости смотрите детали в `GOOGLE_PLAY_PUBLICATION_GUIDE.md`
