**Попробуйте:**

4. Использовать все функции приложения

### 3. Проверка логов

**В консоли Xcode ищите:**

✅ **Успешное переключение:**
```
⚠️ Domain milviar.ru is not accessible, switching to alternative URL (IP)
🔐 Using alternative URL: true

```

## 🐛 Диагностика проблем

### Проблема 1: Приложение не переключается на IP

**Симптомы:**
- В логах нет `Using alternative URL: true`
- Приложение пытается подключиться к `milviar.ru`

**Решение:**
- Проверьте, что версия билда **19**
- Убедитесь, что используете мобильный интернет (не Wi-Fi)
- Перезагрузите приложение

### Проблема 2: Ошибка SSL при подключении к IP

**Симптомы:**
- `SSL certificate problem` в логах
- Предупреждение о сертификате

**Решение:**
- Это нормально для IP адреса - сертификат выдан для домена
- iOS может показать предупреждение, но подключение должно работать
- Проверьте, что IP адрес добавлен в `Info.plist`

### Проблема 3: Все еще не работает

**Что проверить:**
1. Версия приложения (должна быть 19+)
2. Используете ли мобильный интернет Мегафона (не Wi-Fi)
3. Открывается ли `https://83.166.246.225` в Safari
4. Логи в консоли Xcode
